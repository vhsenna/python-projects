# Quiz Game

## Sobre
Quiz com cerca de 300 perguntas verdadeiro/falso.

## Como funciona
Um quiz de perguntas e respostas sobre variados temas onde são mostradas dez perguntas ao usuário. Ao final é exibido o placar com o número de acertos.
